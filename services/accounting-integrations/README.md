# Accounting Integrations Service

OAuth integration and data synchronization for QuickBooks, Xero, and NetSuite.

## Supported Systems

✅ **QuickBooks Online** - OAuth 2.0
✅ **Xero** - OAuth 2.0  
✅ **NetSuite** - Token-Based Authentication (TBA)

## Quick Setup

See full documentation in service directory.

## Authentication Methods

| Provider | Method | Token Expiry |
|----------|--------|--------------|
| QuickBooks | OAuth 2.0 | 60 min |
| Xero | OAuth 2.0 | 30 min |
| NetSuite | TBA (OAuth 1.0a) | Never |

## Features

- Trial balance import
- Chart of accounts sync
- Journal entry creation
- Automatic token refresh
- Encrypted credential storage
