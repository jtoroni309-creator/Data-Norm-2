## Image Specifications

### Formats
- Use **WebP** for modern browsers with **PNG/JPG fallbacks**
- Use **SVG** for logos and icons when possible
- Optimize all images with tools like ImageOptim or Squoosh

### Sizes
- Hero images: 1920x1080px (16:9 ratio)
- Feature images: 800x600px
- Screenshots: Actual size with 2x retina versions
- Thumbnails: 400x300px
- Icons: 64x64px, 128x128px

### Optimization
- Compress all images before uploading
- Use appropriate formats (WebP > PNG > JPG)
- Include alt text for accessibility
- Use lazy loading for below-the-fold images

## Design Guidelines

### Brand Colors
- Primary: #0ea5e9 (Sky Blue)
- Accent: #d946ef (Purple)
- Gradients: Use linear gradients from primary to accent

### Style
- Modern, clean, professional
- Use white space generously
- Consistent with tech/SaaS aesthetic
- Show real product interfaces when possible

### Tools for Creating Images
- **Figma** - Design mockups and interfaces
- **Canva** - Marketing graphics and social media images
- **Photoshop** - Photo editing
- **Illustrator** - Vector graphics and logos
- **Sketch** - UI/UX design

## To Convert SVG to PNG/ICO

Use online tools or command-line utilities:

```bash
# Using ImageMagick
convert favicon.svg -resize 32x32 favicon-32x32.png
convert favicon.svg -resize 16x16 favicon-16x16.png
convert favicon.svg -resize 180x180 apple-touch-icon.png

# Using online tools
# - https://realfavicongenerator.net/
# - https://www.favicon-generator.org/
```
